# Number-translator

### How to use:
```javascript
    const { translateNumber } = require("number-translator");
    
    console.log(translateNumber(123)) // —> "сто двадцать три"
    console.log(translateNumber("123")) // —> "сто двадцать три"
```
###### And that' all!
